This theme ships `TradingViewChart`, built on the free [TradingView Advanced Chart](https://www.tradingview.com/widget-docs/widgets/charts/advanced-chart/) widget. Default style is **candlesticks (K-line)**.

## Bitcoin · 1H

<TradingViewChart symbol="BINANCE:BTCUSDT" interval="60" locale="en" />

## Apple · Daily

<TradingViewChart symbol="NASDAQ:AAPL" interval="D" locale="en" />

## Usage in MDX

```mdx
import TradingViewChart from '../../../components/TradingViewChart.astro';

<TradingViewChart symbol="BINANCE:BTCUSDT" interval="60" locale="en" />
```

Key props: `symbol` (required), `interval` (`1`/`5`/`60`/`D`/`W`), `locale`, `style` (`1` = candles). The chart area keeps a **4:3** aspect ratio from the container width. Charts remount when the site theme toggles.
