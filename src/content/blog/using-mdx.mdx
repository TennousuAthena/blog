## 使用建议

- 叙述与论证继续用 Markdown 语法
- 只有需要组件时才写成 `.mdx`
- 交互组件记得加客户端指令（如需要 hydration）

更多语法见 [MDX 文档](https://mdxjs.com/docs/what-is-mdx)。
